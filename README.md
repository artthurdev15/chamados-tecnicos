# 🛠️ Sistema de Chamados Técnicos

## 📖 Sobre o Projeto
O **Chamados Técnicos** é um sistema focado em otimizar e organizar o fluxo de pedidos de suporte e manutenção de infraestrutura de TI. A ideia principal é facilitar o registro de problemas e o acompanhamento de tudo que precisa ser resolvido pela equipe técnica, saindo da desorganização e centralizando os tickets em um só lugar.

## ✨ O que o sistema faz
* **Abertura de Chamados:** Criação de tickets detalhando problemas de hardware, rede ou sistemas.
* **Acompanhamento de Status:** Controle visual para saber se o chamado está "Aberto", "Em Andamento" ou "Resolvido".
* **Organização Técnica:** Direcionamento dos problemas para a equipe de infraestrutura.
* **Histórico de Soluções:** Registro de como as falhas foram resolvidas para facilitar o trabalho no futuro.

*(Nota: Ajuste caso tenha usado outras ferramentas)*

* **Banco de Dados:** MySQL / PostgreSQL
* **Frontend:** HTML, CSS e JavaScript
